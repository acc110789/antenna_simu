### Antenna Simulation
本程序用于无线局域网MAC层网路包传输过程仿真,另有`Smpl`程序也可以实现无线局域网络的MAC层仿真,
但`Smpl`基于面向过程采用事件分发的方式编程,整个仿真过程晦涩不易看懂.<br/>
本程序采用面向对象的方式重新设计程序.整个仿真过程跟真实场景的过程基本一致.<br/>
`master`分支是对全向天线技术的仿真<br/>
`direct_antenna`分支用于定向天线技术的仿真.
