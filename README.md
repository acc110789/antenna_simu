### antenna_simu
本程序实现了无线局域网MAC层仿真,另有smpl程序也可以实现无线局域网络的MAC层仿真,
但smpl面向过程采用event分发的方式,整个仿真过程晦涩不易看懂.本程序采用面向对象
的方式重新设计程序.整个仿真过程跟真实场景的过程基本一致.
